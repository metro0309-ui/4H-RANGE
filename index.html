백테스트 엔진 3개월 · 4H
검색 시 Binance에서 3개월치 4H 캔들(약 540봉) 로드.
각 봉에서 Low+X% 매수 → 다음 3봉(12시간) 내 목표 도달 여부 시뮬레이션.
버퍼 3~10% 중 승률 최고값을 자동 선택해 매수가에 적용.
매도 3단계 공식 선반영
Range = High − Low
BUY  = Low + (Range × 최적버퍼%)
TP1  = Low + (Range × 0.382) // Fib 38.2%
TP2  = Low + (Range × 0.618) // 황금비 61.8%
TP3  = High − (Range × 0.07) // −7% 고점 근처
STOP = Low − (Range × 0.05) // −5% 하방
API 호출 제한 하루 6회
4시간봉 원칙 — 하루 6회 갱신에 맞춰 API도 6회 제한.
자정(00:00)에 자동 리셋. 수동 High/Low 입력은 제한 없음.
